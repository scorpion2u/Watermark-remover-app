Watermark Editor (safe)
This sample app is a simple image editor for learning and legitimate use.
It intentionally includes an "Add Watermark" action (to help users mark ownership).
It does NOT provide any functionality to remove or bypass watermarks.

Files:
- main.py : Kivy app
- buildozer.spec : Buildozer config

To build: use buildozer in a proper environment (or CI).
